Step 2 · Install the SDK

Step 3 · Point the CLI at this server

Paste the key from step 1. This server's endpoint is prefilled.

ql login --token <your-api-key> --endpoint {{ endpoint }}

ql login verifies the token immediately. Hitting connection trouble? Run ql doctor — it pinpoints whether it's the endpoint, your network, the token, or a missing connector.

Step 4 · Capture a run & push it

import quantumledger as ql
from qiskit import QuantumCircuit
from qiskit_aer import AerSimulator

@ql.capture(project="bell")
def run_bell():
    qc = QuantumCircuit(2)
    qc.h(0); qc.cx(0, 1); qc.measure_all()
    return AerSimulator().run(qc, shots=4096)

run_bell()
ql push          # syncs local runs here (idempotent)

Then your runs appear under Records.
