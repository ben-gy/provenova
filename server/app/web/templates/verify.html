What this proves

An {{ term("attestation") }} is an Ed25519 signature over a Merkle root of the compliance evidence collected from your runs at a point in time. Anyone can verify it against the issuer's public keys with no privileged access. If any referenced record is altered after signing, Evidence intact fails; if the issuer revokes it, Not revoked fails.
